# ability-test

#お問い合わせフォーム

##環境構築

Dockerビルド

1.ディレクトリの作成  
2.docker-compose.ymlの作成  
3.Nginxの設定  
4.PHPの設定  
5.MySQLの設定  
6.phpMyAdminの設定  
7.docker-compose up -d —build  

Laravel環境構築

1.docker-compose exec php bash  
2.composer install  
3.php artisan migrate  

##使用技術
